Introduction aux didacticiels du tableau de bord GBADs

Buts d'apprentissage​

À la fin de ces didacticiels, vous aurez appris à créer un tableau de bord à l'aide des composants Plotly Dash dans Python 3. Vous aurez également appris la conception de tableaux de bord.

Conditions préalables​

  • Une connaissance de base de la programmation en Python 3 est utile, mais l'installation et la configuration de l'environnement Python 3 sont expliquées ci-dessous.
  • La connaissance du langage HTML est utile car Dash utilise également des composants HTML courants.
  • La connaissance de CSS est utile car les composants Dash peuvent être stylisés avec CSS.

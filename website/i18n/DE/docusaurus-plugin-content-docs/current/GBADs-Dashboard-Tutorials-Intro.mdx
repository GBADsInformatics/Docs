## Lernziele
Am Ende dieser Tutorials haben Sie gelernt, wie man ein Dashboard mit Plotly Dash-Komponenten in Python 3 erstellt. Sie haben auch etwas über das Dashboard-Design gelernt.
## Voraussetzungen
<ul>
    <li>Ein grundlegendes Wissen in der Programmierung in Python 3 ist nützlich, jedoch wird die Installation und Einrichtung von Python 3 unten erläutert.</li>
    <li>Kenntnisse in HTML sind nützlich, da Dash auch gebräuchliche HTML-Komponenten verwendet.</li>
    <li>Kenntnisse in CSS sind nützlich, da Dash-Komponenten mit CSS gestaltet werden können.</li>
</ul>

## Objetivos de Aprendizaje
Al final de estos tutoriales, habrás aprendido cómo crear un panel de control utilizando componentes de Plotly Dash en Python 3. También habrás aprendido sobre el diseño de paneles de control.

## Requisitos Previos
(No se han traducido los elementos HTML y JavaScript)
<ul>
    <li>Es útil tener un conocimiento básico de programación en Python 3, sin embargo, se explica la instalación de Python 3 y la configuración del entorno a continuación.</li>
    <li>Es útil tener conocimiento de HTML, ya que Dash también utiliza componentes HTML comunes.</li>
    <li>Es útil tener conocimiento de CSS, ya que los componentes Dash se pueden diseñar con CSS.</li>
</ul>

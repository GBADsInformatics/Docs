## Objectifs d'apprentissage
À la fin de ces tutoriels, vous aurez appris comment créer un tableau de bord en utilisant les composants Plotly Dash en Python 3. Vous aurez également acquis des compétences en matière de conception de tableaux de bord.

## Prérequis
<ul>
    <li>Une connaissance de base de la programmation en Python 3 est utile, cependant, l'installation de Python 3 et la configuration de l'environnement sont expliquées ci-dessous.</li>
    <li>La connaissance de HTML est utile, car Dash utilise également des composants HTML courants.</li>
    <li>La connaissance de CSS est utile, car les composants Dash peuvent être stylisés avec CSS.</li>
</ul>

Dash est relativement simple à utiliser et à apprendre. Il fonctionne également bien avec d'autres modules Python. Il offre également plus de flexibilité que d'autres bibliothèques de tableaux de bord similaires.
